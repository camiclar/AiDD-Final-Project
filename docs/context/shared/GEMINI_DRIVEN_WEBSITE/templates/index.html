üìã How to Use This App

  1. API Key: Enter your Google Gemini API key. You can get one for free at Google AI Studio. If you've set the GEMINI_API_KEY environment variable, you can leave this blank.
  2. Model: Select which Gemini model you want to use from the dropdown. Different models have different capabilities and speed.
  3. Prompt: Type your question or instruction in the text area. Be specific for better results!
  4. Submit: Click "Ask model" to send your prompt to the Gemini API and see the response.
  5. History: All successful requests are saved to a database. Click "View generation history" below to see past conversations.
